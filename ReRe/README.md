# ReRe Project Website

This folder contains the project page for:

```
Reason, Then Re-reason: Cross-view Revisiting Improves Spatial Reasoning
```

The page is a static GitHub Pages site adapted from the SaFiRe/Nerfies-style project template.

## Structure

- `index.html`: main project page.
- `static/images/`: figures copied or converted from the ReRe paper source.
- `static/_Camera_Ready__Reason__Then_Re_reason__Cross_view_Revisiting_Improves_Spatial_Reasoning.pdf`: local paper PDF linked from the page.
- `static/css/` and `static/js/`: website template dependencies.
